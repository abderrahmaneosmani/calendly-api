# calendly-api
calendly api





## Installation

```bash
git clone https://github.com/abderrahmaneosmani/calendly-api.git
cd calendly-api
$ npm install

```
## Add .env file
```bash
DATABASE_URL="mysql://root:@localhost:3306/test?schema=public"


```


## Running the app

```bash
# development
$ npm run dev
```

## Running the test
```bash
# development
$ npm run test
```
